¿Qué es un administrador de contraseñas?

El Administrador de Contraseñas es un sistema que guarda sus distintas cuentas (usuario y contraseña) de multiples sitios web en una misma cuenta. La base de datos donde se guarda esta informacion esta cifrada mediante una unica clave de forma que el usuario solo tenga que memorizar una clave para acceder a todas las demas. Esto facilita la administracion de contraseñas y fomenta que los usuarios escoja claves complejas sin miedo a que no podran recordarlas posteriormente
